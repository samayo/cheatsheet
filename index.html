NGINX

Update and Install NGINX

sudo apt update
sudo apt upgrade -y
sudo apt install nginx -y

Start and Enable NGINX

sudo systemctl start nginx
sudo systemctl enable nginx

Configure NGINX

sudo vim /etc/nginx/nginx.conf
sudo nginx -t
sudo systemctl reload nginx

PHP

Install PHP 8.2 and Extensions

sudo apt update
sudo apt install php8.2 php8.2-fpm -y
sudo apt install php8.2-common php8.2-mysql php8.2-xml php8.2-xmlrpc php8.2-curl php8.2-gd php8.2-imagick php8.2-cli php8.2-dev php8.2-imap php8.2-mbstring php8.2-opcache php8.2-soap php8.2-zip php8.2-intl -y

Check PHP Version

php -v

Edit PHP Configuration (Optional)

sudo vim /etc/php/8.2/fpm/php.ini
sudo systemctl restart php8.2-fpm

MariaDB

Install and Secure MariaDB

sudo apt update
sudo apt install mariadb-server -y
sudo mysql_secure_installation

Start and Enable MariaDB

sudo systemctl start mariadb
sudo systemctl enable mariadb

Access MariaDB Shell

sudo mysql -u root -p

Utils

Install Common Utilities

sudo apt update
sudo apt install wget git curl zip unzip net-tools -y

Let's Encrypt

Install Certbot

sudo apt update
sudo apt install snapd -y
sudo snap install core; sudo snap refresh core
sudo snap install --classic certbot
sudo ln -s /snap/bin/certbot /usr/bin/certbot

Obtain and Manage Certificates

sudo certbot --nginx
sudo certbot --nginx -d yourdomain.com -d www.yourdomain.com
echo "0 12 * * * /usr/bin/certbot renew --quiet"

SSH Configuration

Install and Configure SSH

sudo apt update
sudo apt install openssh-server -y
sudo systemctl status ssh
sudo vim /etc/ssh/sshd_config
sudo systemctl restart ssh
sudo systemctl enable ssh

MySQL Commands

Database and User Management

CREATE DATABASE your_database_name;
CREATE USER 'your_username'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON your_database_name.* TO 'your_username'@'localhost';
FLUSH PRIVILEGES;
DROP USER 'username'@'hostname';

Import Database from File

mysql -u username -p database_name < file.sql

Composer Commands

Common Composer Commands

composer create-project vendor/package - Create a new project
composer require vendor/package - Require a package
composer update - Update all packages
composer remove vendor/package - Remove a package
composer install - Install dependencies
composer show - Show installed packages
composer validate - Validate composer.json
composer search keyword - Search for packages
composer dump-autoload -o - Optimize autoload
composer run-script scriptname - Run scripts

Publishing NPM Library

Versioning and Publishing

npm version [patch|minor|major] - Updates package.json version
git tag {tag.version.number} -m "new entry" - Update git tag
git push origin {tag.version.number} - Push tag
npm login - Login to NPM
npm publish - Publish to NPM
